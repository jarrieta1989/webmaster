PRESENTACIÓN DEL PROYECTO: 
El objetivo principal de este proyecto es el análisis, diseño y construcción de una aplicación de software para controlar las ventas físicas y virtuales 
de una empresa de manufactura. El proyecto se realizará siguiendo la metología SCRUM, y se entregarán 5 sprints durante su desarrollo. 
Nuestro equipo se enfocará en aportar una solución funcional, agradable al usuario y acorde a las necesidades de la empresa interesada. 
Se estima que el tiempo de desarrollo inicial es de 8 semanas, y la aplicación web contará con 5 módulos principales:
  1. Módulo para la administración de ventas
  2. Módulo para registrar el estado de las ventas
  3. Módulo para la gestión de los vendedores
  4. Módulo para la gestión de ingreso al sistema de información
  5. Módulo de gestión de usuarios y roles
Para el desarrollo del proyecto se tendrá en cuenta un análisis del usuario y sus necesidades, la implementación de soluciones prácticas y acordes al problema, 
asi como la integración de los conocimientos obtenidos en el proceso de formación MinTic. Como equipo buscamos el desarrollo de una aplicación web que cumpla 
las expectativas del usuario, y que nos permita poner en práctica toda nuestra creatividad y las habilidades adquiridas en este proceso de aprendizaje.
